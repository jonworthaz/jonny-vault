# Base Reality — Daily Video Generation Prompt

*A reusable engine for generating daily, high-definition, vertical kinetic-text
videos with moving ASMR sound for **base-reality.com** social channels.*

> **Objective:** the most beautiful, colourful, dynamic, and original short-form
> video we can generate — every day, on theme, without repeating ourselves.
> Reference look: the flowing, high-def animated typography on a site like
> vertus.ai — but in **9:16**, brand-coloured, and with native sound.

---

## How to use this

1. Pick the **video model** — use one with **native audio** so the ASMR comes
   out of the same generation (e.g. Veo 3 / Sora 2). Audio-less models (Runway,
   Kling, Luma) work too; add the sound in step 4.
2. Choose today's **theme** from the [7-day rotation](#the-7-day-rotation) (or
   spin a new one) and today's **line of text** (the hook).
3. Paste the [Master Prompt](#master-prompt) and fill the `{{slots}}`.
4. If your model has no audio, layer the [ASMR audio recipe](#asmr-audio-moving-sound)
   in any editor (CapCut/Premiere) — keep the whooshes synced to the motion.
5. Export **9:16, 1080×1920 min (4K if available), 24–30 fps, ~8–15 s**.

## Master Prompt

> Copy this whole block. Replace every `{{slot}}`. Keep the **STYLE LOCK** and
> **AUDIO** sections identical day to day — only the theme, text, and palette
> change. That consistency is the brand; the variation is the content.

```
VERTICAL 9:16 KINETIC-TYPOGRAPHY VIDEO — ULTRA HIGH DEFINITION, ~{{8–12}} seconds.

CONCEPT (today): {{one-line theme — e.g. "the moment the simulation resolves into base reality"}}
ON-SCREEN TEXT (appears in beats): "{{your daily line — short, 3–8 words}}"
BRAND: Base Reality — base-reality.com

STYLE LOCK (keep constant):
- Cinematic, hyper-real, dreamlike. Think liquid light, volumetric glass,
  holographic depth, soft fractal bloom — premium and otherworldly, never busy.
- Crisp variable-weight sans-serif typography that feels alive: letters
  assemble, breathe, refract, and dissolve like light through water.
- Shallow depth of field, gentle parallax, slow push-in camera, subtle bokeh.
- Immaculate motion: ease-in-out, no jitter, buttery 24–30 fps, motion blur.
- Negative space and restraint — one hero idea on screen at a time.

COLOUR (today): {{palette — e.g. "iridescent magenta→cyan gradient with gold sparks"}}
- Saturated but elegant; gradients that shift over time; luminous highlights.

MOTION BEATS:
1. Open on darkness/void; a single point of light blooms.
2. The text materialises letter-by-letter from flowing particles/liquid light.
3. Camera pushes in; colour washes across; the scene "resolves" into clarity.
4. Text holds, glows, then dissolves back into light. Loop-friendly ending.

CAMERA: slow dolly-in with micro-drift; one elegant move, no cuts.
RENDER: 4K, ultra-sharp, photoreal lighting, ray-traced reflections, film grain
        kept to a whisper. Aspect 9:16. NO watermark, NO stock-template look.

AUDIO (native ASMR, spatial/moving):
- Soft binaural ambience that PANS left↔right as the camera and light move.
- A gentle rising "whoosh" synced to each letter forming; a warm, deep
  sub-bass swell on the camera push-in; delicate crystalline tings/shimmer as
  text glows; airy tail as it dissolves. Calm, tingly, premium. No music vocals,
  no voiceover. Quiet, intimate, headphone-first.

AVOID: clutter, stocky corporate vibes, harsh cuts, ugly default fonts,
       low-contrast mud, more than one message on screen, anything generic.
```

## The 7-day rotation

Rotate the **theme + palette + motion metaphor** so the feed never feels
repetitive. Same STYLE LOCK + AUDIO every day = recognisable brand; new theme
every day = fresh content.

| Day | Theme (concept) | Palette | Motion metaphor |
|---|---|---|---|
| Mon | "Wake up inside base reality" | Iridescent magenta → cyan, gold sparks | Glitch → resolves to crystal clarity |
| Tue | "The simulation is optional" | Deep indigo → electric violet, silver | Pixels dissolving into liquid light |
| Wed | "Presence is the real frontier" | Warm amber → rose gold, cream | Slow bloom, breathing light |
| Thu | "Consciousness is the substrate" | Emerald → teal → aqua, white glow | Fractal unfolding, holographic depth |
| Fri | "Render your own world" | Sunset coral → tangerine → fuchsia | Wireframe → fully rendered scene |
| Sat | "Quiet the noise, see the signal" | Monochrome blue with one neon accent | Static → single clean beam of light |
| Sun | "You are the base layer" | Full-spectrum prism / aurora | Aurora ribbons weaving the text |

> Keep a running list of used lines so you never repeat a hook. Each day = one
> new line, one theme row, one generation.

## ASMR audio (moving sound)

If your video model generates audio natively, the AUDIO block above is enough.
If not, build this layer in your editor and sync to the motion:

- **Bed:** soft binaural room tone / airy pad, gently **panning L↔R** with the
  camera drift (this is the "moving" part — it should feel like it travels).
- **Per letter:** a tiny rising *whoosh* / paper-soft swish as each word forms.
- **Push-in:** a warm low **sub-bass swell** (felt more than heard).
- **Glow hold:** delicate **crystalline shimmer / tings**, like glass chimes.
- **Dissolve:** an airy reverse-reverb **tail** that fades to silence.
- Mix calm and intimate — **headphone-first**, no music, no voice. Loudness
  low; let the texture do the work.

## Social packaging (per post)

- **Caption:** lead with the on-screen line, then one reflective sentence.
- **Hook in first 0.5 s:** the light-bloom + first letter must land instantly.
- **Loop:** end on light dissolving back to the opening void so it loops clean.
- **Hashtags:** brand + concept (e.g. #BaseReality #SimulationTheory #ASMR
  #KineticType #DigitalArt) — tune per platform.
- **Cadence:** one per day, following the rotation; archive used lines.
